# Flomo Export Helper Rule

开发一个chrome 插件，在 flomo 网页版中，在每条笔记的标题栏添加一个复选框，复选框样式美观，具有悬停效果，选中时显示绿色对勾，可以通过复选框选择要导出的笔记。

## 功能说明
1. 在每条 flomo 笔记的标题栏添加一个复选框
2. 复选框样式美观，具有悬停效果
3. 选中时显示绿色对勾
4. 可以通过复选框选择要导出的笔记

## 使用方法
1. 将此规则添加到 Cursor 插件中
2. 访问 flomo 网页版时会自动在每条笔记后添加导出选择框
3. 选中需要导出的笔记
4. 可以通过开发其他功能来实现批量导出选中的笔记

## 注意事项
1. 确保选择器匹配 flomo 最新版本的 DOM 结构
2. 样式可能需要根据 flomo 的主题进行调整
3. 建议添加导出按钮和导出功能的实现
